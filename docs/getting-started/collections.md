HVE Core ships as focused VS Code extension editions. Each collection targets a specific domain — install only what your team needs, or get everything with the Full collection.

> [!TIP]
> Not sure which to pick? Install the **Full** collection (`ise-hve-essentials.hve-core`) to get every stable artifact across all domains, then narrow down later.

## Collection Details

<details>
<summary><strong>Full</strong> — All stable artifacts across every domain</summary>

<br>

HVE Core provides the complete collection of AI chat agents, prompts, instructions, and skills for VS Code with GitHub Copilot. This edition includes every artifact across all domains: development workflows, architecture, Azure DevOps, data science, design thinking, security, and more. Use this edition when you want access to everything without choosing a focused collection.

**What's included:** 36 agents · 45 prompts · 75 instructions · 3 skills

**Install:** [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=ise-hve-essentials.hve-core)

</details>

<details>
<summary><strong>HVE Core Workflow</strong> — RPI workflow and Git prompts</summary>

<br>

HVE Core provides the flagship RPI (Research, Plan, Implement, Review, Discover) workflow for completing complex tasks through a structured five-phase process. The RPI workflow dispatches specialized agents that collaborate autonomously to deliver well-researched, planned, and validated implementations. This collection also includes Git workflow prompts for commit messages, merge operations, repository setup, and pull request management.

**What's included:** 17 agents · 15 prompts · 7 instructions · 1 skill
